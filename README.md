Основная дисциплина модуля: Технология разработки и защиты баз данных
 
Задание 1 Проектирование базы данных
 
1
Спроектировать структуру базы данных по тематике дипломного проекта 
• определить таблицы;
• задать поля и типы данных;
• определить первичные и внешние ключи;
• описать связи между таблицами.
 
2
Реализовать базу данных в выбранной СУБД (MySQL / PostgreSQL / SQLite и др.).
 
3
Заполнить данными таблицы БД.
 
4
Подготовить и выполнить не менее 5 запросов:
SELECT (с условием);
INSERT;
UPDATE;
DELETE;
SELECT с JOIN (если есть несколько таблиц).
